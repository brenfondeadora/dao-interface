## 💻 Run Locally

**Step 1:**

Install dependencies `yarn install`

**Step 2:**

Run the app on localhost with `yarn start`

**Step 3:**

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

**Step 4:**

To create build files `yarn build`
